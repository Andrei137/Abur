# Abur

## Setup

1. Clone the repository
```sh
git clone https://github.com/Andrei137/Abur
```

2. Change the current working directory
```sh
cd Abur
```

3. Install the dependencies
```sh
pnpm install
```

4. Create the database
##### Windows
```sh
pnpm run db:win
```

##### Linux
```sh
pnpm run db:linux
```

5. Start the server
```sh
pnpm run start
```
